# Titanic Data Analysis 

##  Project Overview
This project analyzes survival patterns from the Titanic dataset using Python.

##  Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

##  Analysis Performed
- Survival comparison by gender
- Survival comparison by passenger class
- Survival rate by age group
- Data cleaning and visualization

##  Files
- Titanic_Data_Analysis_Task2_FIXED.ipynb – Main notebook
- titanic.csv – Dataset

##  Key Insights
- Females had a higher survival rate than males
- First-class passengers survived more
- Children had better survival chances

## Dataset Source
Kaggle Titanic Dataset
